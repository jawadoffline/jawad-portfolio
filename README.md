# jawad-portfolio

My personal portfolio — built as a **Spring Boot application**, not a static page. The portfolio IS the proof.

**Live:** [jawadali.dev](https://jawadali.dev)

## Tech Stack

| Layer | Tech |
|-------|------|
| Backend | Java 21, Spring Boot 3.2, Thymeleaf |
| Frontend | HTMX, Tailwind CSS, Vanilla JS |
| Real-time | Server-Sent Events (SSE) |
| Reverse Proxy | Caddy (auto HTTPS via Let's Encrypt) |
| Hosting | Oracle Cloud (Always Free tier) |
| CI/CD | GitHub Actions (auto-deploy on push) |
| Domain | Cloudflare DNS |

## Features

### Interactive

- **Live Terminal** — Embedded terminal where every command (`whoami`, `ls projects/`, `neofetch`, `cat about.txt`, `java --version`, `uptime`) hits a Spring Boot controller and returns server-rendered HTML.

- **"Solve With Me" Walkthrough** — Step-by-step interactive walkthroughs of real production problems: debugging a frozen Swing UI, hunting a memory leak in mission-critical GCS software, and designing the BondLens architecture. Three scenarios with code snippets.

- **Command Palette** — Press `/` or `Ctrl+K` to open a VS Code-style command palette for navigation, theme switching, and easter eggs.

- **Matrix Rain** — Type `matrix` in the terminal to toggle a Matrix-style green character cascade behind the terminal.

### Real-time

- **Live Visitor Counter** — SSE streams the active visitor count to all connected browsers. Open two tabs and watch it update.

- **Persistent Visit Counter** — Total site visits tracked server-side, persisted to disk, displayed in footer.

### Content

- **Boot Sequence** — Terminal-style intro animation on first visit (skippable, session-cached).

- **Career as Mission Log** — Timeline with live status badges, tech tags, and pulsing indicators for current role.

- **Projects as Case Studies** — Situation / Action / Result format instead of generic project cards.

- **Skills as System Diagnostics** — Animated progress bars with status labels (PRIMARY, ADVANCED, VETERAN).

- **Real Code Showcase** — Actual bond pricing method from BondLens with syntax highlighting. Not "I know Java" — here IS my Java.

- **GitHub-style Contribution Heatmap** — 20-week coding activity grid with realistic patterns.

- **"What I'm Building Next" Roadmap** — Three cards showing current learning trajectory (Cloud-Native, Reactive Systems, System Design).

### Practical

- **HTMX Contact Form** — Submits via HTMX to a Spring Boot endpoint with rate limiting and input validation. No page reload.

- **vCard Download** — One-click `.vcf` contact file generated by a REST endpoint.

- **Server-Rendered Resume** — `/resume` renders a print-optimized HTML resume via Thymeleaf with "Save as PDF" button.

- **Dark/Light Theme** — Full theme toggle with localStorage persistence.

- **Scroll Progress Bar** — Emerald gradient bar tracking scroll position.

- **Typing Animation** — Hero subtitle cycles through roles: "Building Trading Platforms", "Engineering Defense Systems", etc.

- **Page Load Time** — Footer displays actual page load time in milliseconds.

## Security

- Security headers (X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy)
- Contact form rate limiting (5 per IP) and input length validation
- SSE connection limits (max 50) with 5-minute timeout
- Caddy handles HTTPS with auto-renewing Let's Encrypt certificates
- No secrets in repository — GitHub Actions secrets for deployment

## Run Locally

```bash
# Requires Java 21 and Maven
mvn spring-boot:run
# Open http://localhost:8080
```

## Deployment

Auto-deploys on every `git push` via GitHub Actions:

```
git push → GitHub Actions builds JAR → SCP to Oracle VM → restart service
```

### Manual / Docker

```bash
# Docker
docker build -t jawad-portfolio .
docker run -p 8080:8080 jawad-portfolio

# Direct
mvn clean package -DskipTests
java -jar target/portfolio-1.0.0.jar
```

### Infrastructure

- **Oracle Cloud** — Always Free VM (1 OCPU, 1 GB RAM)
- **Caddy** — Reverse proxy with automatic HTTPS
- **GitHub Actions** — CI/CD pipeline (build + SCP + restart)
- **Cloudflare** — DNS management
