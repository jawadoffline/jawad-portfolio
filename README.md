# jawad-portfolio

My personal portfolio — built as a **Spring Boot application**, not a static page.

**Live:** [jawadali.up.railway.app](https://jawadali.up.railway.app)

## Tech Stack

| Layer | Tech |
|-------|------|
| Backend | Java 21, Spring Boot 3.2, Thymeleaf |
| Frontend | HTMX, Tailwind CSS (CDN), Vanilla JS |
| Real-time | Server-Sent Events (SSE) |
| Deployment | Docker, Railway |

## What Makes This Different

This isn't a template or a static HTML page. It's a running Spring Boot application where every interactive feature hits a real server endpoint.

- **Live Terminal** — An interactive terminal embedded in the page. Every command (`whoami`, `ls projects/`, `neofetch`, `cat about.txt`) is processed by a Spring Boot controller and returns server-rendered HTML via HTMX.

- **Real-time Visitor Counter** — Server-Sent Events stream the live visitor count to all connected browsers. Open two tabs and watch it update.

- **"Solve With Me" Walkthrough** — Interactive step-by-step walkthroughs of how I approach real production problems: debugging a frozen Swing UI, hunting a memory leak in mission-critical software, and designing a clean architecture.

- **Command Palette** — Press `/` or `Ctrl+K` anywhere to open a VS Code-style command palette for navigation.

- **Matrix Rain** — Type `matrix` in the terminal to toggle a Matrix-style character cascade.

- **HTMX Contact Form** — Form submits via HTMX to a Spring Boot endpoint. No page reload, no JavaScript framework.

- **Server-Generated Resume** — `/resume` renders a print-optimized HTML resume via Thymeleaf. Click "Save as PDF" to download.

- **vCard Download** — One-click `.vcf` contact file generated by a REST endpoint.

- **Dark/Light Theme** — Full theme toggle with localStorage persistence.

- **3 Easter Eggs** — Find them all.

## Run Locally

```bash
# Requires Java 21 and Maven
mvn spring-boot:run
# Open http://localhost:8080
```

## Deploy

The app is containerized and deploys anywhere that runs Docker or Java:

```bash
# Docker
docker build -t jawad-portfolio .
docker run -p 8080:8080 jawad-portfolio

# Railway (auto-deploys on git push)
# Uses railway.toml + Dockerfile
```
